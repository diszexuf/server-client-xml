# XML Messaging System

Клиент-серверное приложение на Java для обмена сообщениями в формате XML с автоматической
фильтрацией контента и логированием в базу данных

### Стек технологий
* **Java 21**
* **XmlBeans 5.3.0**
* **JDBC + SQLite**
* **Gradle (Kotlin)**
* **JUnit 5**

### Запуск проекта

Проект содержит скрипты, которые сами генерируют XML-классы и запускают окружение

**Windows:**
```bash
run.bat
```

**Linux:**
```bash
chmod +x run.sh
./run.sh
```
Скрипт запустит одно окно сервера и три окна клиентов (количество клиентов можно изменять)

### Команды клиента

* `-m <текст>` — отправить сообщение
* `-h` — выход из приложения
